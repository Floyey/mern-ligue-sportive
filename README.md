# mern-ligue-sportive

## Download

```sh
git clone https://github.com/Jeremy-BD/mern-ligue-sportive.git
cd mern-ligue-sportive
```

## Installation

Create .env file with connection parameters :
```js
DATABASE_USER=""
DATABASE_PASSWORD=""
```

### Back-end

```sh
cd back
npm install
npm run server
```

### Front-end

```sh
cd front
npm install
npm start
```
